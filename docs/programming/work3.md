## Формулировка задания

Необходимо разработать программу на языке Python, которая строит бинарное дерево. Бинарное дерево — это структура данных, в каждом узле которой может быть не больше двух потомков: левого и правого.

Базовый вариант отображения результата — дерево в виде вложенного словаря. Дополнительно необходимо рассмотреть другие структуры данных, в том числе контейнеры из модуля `collections`, которые можно использовать для хранения бинарного дерева.

Также требуется:

- реализовать рекурсивную функцию `gen_bin_tree`;
- учитывать параметры `height` и `root`, переданные в функцию;
- если параметры не переданы, использовать значения из индивидуального варианта;
- добавить документацию к коду в формате docstring;
- использовать аннотации типов;
- написать тесты с использованием модуля `unittest`.

## Используемый вариант

В работе реализован вариант №9:

```text
Root = 9
height = 6
left_leaf = root * 2 + 1
right_leaf = 2 * root - 1
```

То есть по умолчанию дерево строится с корневым значением `9` и высотой `6`.

Для каждого узла значения потомков вычисляются по формулам:

```python
left_leaf = root * 2 + 1
right_leaf = 2 * root - 1
```

## Что было сделано

В проекте реализована программа для построения бинарного дерева по заданным правилам.

Основная функция проекта — `gen_bin_tree`. Она рекурсивно строит дерево, начиная с корневого узла, и для каждого узла вычисляет левого и правого потомка по формулам варианта №9.

Если при вызове функции не передавать параметры, используются значения по умолчанию:

```python
root = 9
height = 6
```

Пример вызова:

```python
from binary_tree import gen_bin_tree

tree = gen_bin_tree()
print(tree)
```

Также функция поддерживает передачу собственных значений `root` и `height`:

```python
from binary_tree import gen_bin_tree

tree = gen_bin_tree(height=3, root=10)
print(tree)
```

В этом случае дерево будет построено не от стандартного корня `9`, а от переданного значения `10`, а высота дерева будет равна `3`.

## Логика работы программы

Программа работает следующим образом:

1. Определяется текущее значение узла.
2. Если высота дерева равна `1`, создаётся только один узел без потомков.
3. Если высота больше `1`, вычисляются левый и правый потомки.
4. Для каждого потомка рекурсивно вызывается построение поддерева.
5. Рекурсия завершается, когда достигнута нужная высота дерева.

Таким образом создаётся полное бинарное дерево, где каждый узел, кроме листьев, имеет двух потомков.

## Представление дерева

Базовый вариант хранения дерева реализован через вложенные словари.

Пример структуры одного узла:

```python
{
    "value": 9,
    "left": {...},
    "right": {...}
}
```

Здесь:

- `value` — значение текущего узла;
- `left` — левое поддерево;
- `right` — правое поддерево.

Если узел является листом, то его левый и правый потомки равны `None`:

```python
{
    "value": 9,
    "left": None,
    "right": None
}
```

## Дополнительные контейнеры

Кроме базового варианта через словарь, в проекте реализована возможность построения дерева с использованием других контейнеров.

Поддерживаются следующие варианты:

| Значение параметра `container` | Структура хранения |
| --- | --- |
| `"dict"` | вложенные словари |
| `"list"` | вложенные списки |
| `"ordered_dict"` | `collections.OrderedDict` |
| `"defaultdict"` | `collections.defaultdict` |
| `"deque"` | `collections.deque` |

Пример использования:

```python
from binary_tree import gen_bin_tree

tree_as_dict = gen_bin_tree(height=3, container="dict")
tree_as_list = gen_bin_tree(height=3, container="list")
tree_as_deque = gen_bin_tree(height=3, container="deque")
```

Это позволяет показать, что бинарное дерево можно хранить не только в словаре, но и в других структурах данных.

## Основные функции

### `gen_bin_tree`

Рекурсивно строит бинарное дерево.

Принимает параметры:

- `height` — высота дерева;
- `root` — значение корневого узла;
- `container` — способ хранения дерева.

Если `height` и `root` не переданы, используются значения варианта №9.

### `calculate_left_leaf`

Вычисляет значение левого потомка по формуле:

```python
root * 2 + 1
```

### `calculate_right_leaf`

Вычисляет значение правого потомка по формуле:

```python
2 * root - 1
```

### `count_nodes`

Подсчитывает количество узлов в построенном дереве.

Эта функция используется в тестах для проверки корректности построения дерева.

## Тестирование

Для проверки программы написаны тесты с использованием стандартного модуля `unittest`.

В тестах проверяется:

- построение дерева по варианту №9;
- использование значений по умолчанию `root = 9` и `height = 6`;
- корректность формул для левого и правого потомка;
- возможность передавать собственные значения `root` и `height`;
- построение дерева высоты `1`;
- подсчёт количества узлов в дереве;
- обработка некорректного значения высоты;
- построение дерева в разных контейнерах: `dict`, `list`, `OrderedDict`, `defaultdict`, `deque`.

Запуск тестов выполняется командой:

```bash
python -m unittest -v
```

