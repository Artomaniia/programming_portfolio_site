# Описание задания

## Тема

Решение простой алгоритмической задачи на Python: поиск двух чисел в списке, сумма которых равна заданному значению.

## Формулировка задачи

Необходимо реализовать функцию, которая получает на вход:

- список целых чисел `nums`;
- целевое число `target`.

Функция должна найти два разных элемента списка, сумма которых равна `target`, и вернуть индексы этих элементов.

### Пример

```python
nums = [2, 7, 11, 15]
target = 9
```

Так как `2 + 7 = 9`, функция должна вернуть:

```python
(0, 1)
```

## Что было сделано

В репозитории реализован класс `Solution`, внутри которого находится метод для решения задачи.

Логика решения следующая:

1. Перебираются элементы списка.
2. Для каждого элемента выполняется поиск второго элемента.
3. Проверяется условие: сумма двух элементов должна быть равна `target`.
4. Дополнительно проверяется, что используются два разных индекса.
5. Если подходящая пара найдена, функция возвращает индексы этих элементов.

Решение использует простой перебор всех возможных пар чисел. Такой подход легко понять и подходит для учебной задачи.

## Проверка решения

Для проверки работы программы в репозитории добавлен файл с тестами `test_module.py`, использующий модуль `unittest`.

В тестах проверяются следующие случаи:

```python
[2, 7, 11, 15], target = 9    # результат: (0, 1)
[3, 3, 3, 3], target = 6      # результат: (0, 1)
[16, 2, 15, 89], target = 105 # результат: (0, 3)
```

Эти тесты проверяют, что функция:

- находит пару чисел с нужной суммой;
- возвращает именно индексы элементов, а не сами элементы;
- корректно работает со списком, где есть одинаковые значения;
- не использует один и тот же элемент дважды.

## Как запустить тесты

Для запуска тестов можно использовать команду:

```bash
python -m unittest test_module.py
```

## Примечание

В текущей версии кода важно, чтобы название метода в файле `first.py` совпадало с названием, которое используется в тестах. Тесты вызывают метод `two_sum`, поэтому для корректного запуска решения метод в классе `Solution` также должен называться `two_sum`.
